## 🚀 СЛЕДУЮЩИЕ ШАГИ

### 1. Проверь контент
Открой http://localhost:5173 → пролистай все секции → скажи что не так.

### 2. Замени фото (если нужно)
Сейчас используются фото-плейсхолдеры из Miss Moscow Beauty.

**Где лежат:**
```
public/assets/images/
├── hero/
├── categories/
├── jury/
├── gallery/
└── partners/
```

### 3. Настрой Telegram-бот
В `.env` добавь токен бота для приёма заявок:
```
VITE_TELEGRAM_BOT_TOKEN=твой_токен
VITE_TELEGRAM_CHAT_ID=твой_chat_id
```

### 4. Деплой на Vercel/Netlify
Когда одобришь — деплою на прод.

## 🔧 КОМАНДЫ

**Запустить dev-сервер:**
```bash
cd ~/.openclaw/workspace/projects/world-russian-beauty-site
npm run dev
```

**Билд для продакшена:**
```bash
npm run build
```

**Preview билда:**
```bash
npm run preview
```

**Остановить сервер:**
```bash
kill $(cat .dev.pid)
```
